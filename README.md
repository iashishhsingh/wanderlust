# 🌍 Wanderlust

## 🚀 Features
- User Authentication
- Create/Edit/Delete Listings
- Image Upload (Cloudinary)
- Reviews & Ratings
- Map Integration

## 🛠 Tech Stack
- Node.js, Express
- MongoDB (Mongoose)
- EJS
- Cloudinary

## ⚙️ Setup
git clone https://github.com/iashishhsingh/wanderlust.git
cd wanderlust
npm install
npm start

## 🔐 Environment Variables
Create a .env file:

ATLASDB_URL=
CLOUD_NAME=
CLOUD_API_KEY=
CLOUD_API_SECRET=
SECRET=
